TheMall, built @ EyeCue Lab

TheMall was a social shopping destination, integrating hundreds of consumers favorite brands under one roof. TheMall is designed for an online window-shopping experience, much like strolling through a physical mall.

TheMall was built on a microservices architecture, with a RestifyJS + MongoDB backend talking JSONAPI to a KoaJS and plain jQuery frontend. I was the main frontend developer for this site, and built a static mock in ExpressJS, then built the final reenvisioned frontend out after the client had a design pivot.

The dev version is visible at mall-dev.eyecue.io

Powur, built @ EyeCue Lab

Power is a network marketing solar sales web app built atop SolarCity's physical API for acquiring customers. Users sign up under other users, send proposals to other people, and earn commissions if those individuals follow through and place solar on their property.

Powur is built on Rails with Postgres, and Angular 1.4 in TypeScript. I worked on this project for two months, building out custom endpoints and Angular components for users to easily view their personal sales statistics.

The live site can be viewed at powur.com

Wagick, built @ EyeCue Lab

Wagick was my intern project at EyeCue Lab, and is a word game inspired by Scrabble and every RPG ever. The goal of the game is to spell words, depending on the color of the tiles that you used to spell the word, you will either damage the opponent, gain experience, or recieve gold.

Wagick was built with Rails, Postgres, and plain jQuery. It has server and client-side checking on action allowability, and a fun set of graphics and animations.

It is hosted at wagick.herokuapp.com
